EclipseSHOE

Playing Eclipse Phase with GUMSHOE

Download as .zip Download as .tar.gz View on GitHub

EclipseSHOE is a combination of the CC-lisenced role-playing game Eclipse Phase and the rule-system GUMSHOE.

Version

The current version of EclipseSHOE is 0.3. You can contribute by playtesting, commenting and giving feedback.
